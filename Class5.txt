Nullable type
- By default value cant be null
  - var a: Int = null (NOT ALLOWED)
- The "?" operator allows null values
  - var a:Int? = null (ALLOWED)
  - var x:List<String?> =  listOf(null, null, null) (list of null elements)
  - var x:List<String>? = null (a null list)
- Null Testing
  > var nullTest : Int? = null
  > nullTest?.inc()
- Elvis Operator
  > var x:Int? = 4
  > var y = x?.toString() ?: " "
  - if x is null, then y gets the ""value
- Double bang operator
    > var x: Int? = null
    > x!!.toString()
    - kotlin.KotlinNullPointerException
